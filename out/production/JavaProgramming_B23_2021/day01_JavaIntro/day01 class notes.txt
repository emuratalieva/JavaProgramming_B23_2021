shortcuts:

		undo: ctrl + z

		copy: Command + c
		paste: Command + v
		undo: command + z

how to create project in IntelliJ:
	File -> new -> Project -> Java -> next -> next -> Project Name & Project Location -> Finish 

preference window of intelliJ:
	windows:
		file -> settings

	Mac:
		Intellij IDE -> preferences


set compiler version:
	windows:
		file -> settings -> Build, execution,deployment -> Compiler -> Java compiler -> set byte code version to 8

	Mac: 
		Intellij IDE -> preferences -> Build, execution,deployment -> Compiler -> Java compiler -> set byte code version to 8
